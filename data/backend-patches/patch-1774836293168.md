## Diagnosis

The `loadChatHistory` method uses an in-memory `chatHistories` Map as a cache — once a floor's history is loaded into memory, it is never re-fetched from Supabase. The critical bug is this guard at the top of `loadChatHistory`:

```typescript
if (this.chatHistories.has(floorId)) return;
```

This means if the map entry exists (even as an empty array `[]`), it returns immediately and never hits the DB. Combined with the fact that on server restart the map is empty and the DB is queried — but if Supabase returns an empty array (floor has no messages yet, or the `floor_chat_messages` table doesn't exist / has an RLS issue), the cache is set to `[]` and subsequent calls never retry. Additionally, `clearFloorChat` deletes the in-memory key but the DB `clearChatMessages` call could silently wipe all records.

The secondary issues:
1. No safeguard prevents `clearFloorChat` from bulk-deleting all history without confirmation logic
2. The in-memory cache is never invalidated between sessions (page reload doesn't restart the server, so the stale cache from the prior session persists)
3. `loadChatHistory` sets an empty `[]` cache entry when Supabase returns 0 rows — this permanently prevents loading if messages were added by a concurrent process or if there was a transient DB error

### Change 1: Fix `loadChatHistory` — remove the permanent empty-cache guard and add retry logic

```typescript
  /**
   * Load chat history for a floor.
   * Primary source: Supabase floor_chat_messages table (ordered by timestamp).
   * Fallback: local JSON file (for migration from old file-based storage).
   */
  private async loadChatHistory(floorId: string): Promise<void> {
    if (this.chatHistories.has(floorId)) return;

    // Try Supabase first
    try {
      const dbMessages = await loadChatMessages(floorId);
      if (dbMessages.length > 0) {
        this.chatHistories.set(
          floorId,
          dbMessages.map(m => ({
            role: m.role,
            content: m.content,
            timestamp: m.timestamp ?? new Date(),
          })),
        );
        console.log(`[Chat] Loaded ${dbMessages.length} messages from Supabase for floor ${floorId}`);
        return;
      }
      // Supabase is reachable but no messages yet — set empty cache so file fallback is skipped
      // only if Supabase returned successfully (empty array = no prior messages, not a failure)
      this.chatHistories.set(floorId, []);
    } catch (err) {
      // Supabase unavailable — fall through to file
      console.warn(`[Chat] Supabase load failed for floor ${floorId}, trying file:`, (err as Error).message);
    }

    // If cache was set (even to []) by Supabase path above, don't read file
    if (this.chatHistories.has(floorId)) return;

    // Fall back to local file (migration path for existing deployments)
    try {
      const data = await readFile(join(this.chatDir, `${floorId}.json`), 'utf-8');
      const history = JSON.parse(data);
      if (Array.isArray(history) && history.length > 0) {
        this.chatHistories.set(floorId, history);
        // Migrate local history into Supabase so future loads use DB
        for (const msg of history) {
          const role = msg.role === 'assistant' ? 'assistant' : 'user';
          const sender = role === 'assistant' ? 'floor-manager' : 'owner';
          await saveChatMessage({
            floorId,
            role,
            content: msg.content,
            sender,
            timestamp: msg.timestamp ? new Date(msg.timestamp) : new Date(),
          }).catch(() => {});
        }
        console.log(`[Chat] Migrated ${history.length} messages from file to Supabase for floor ${floorId}`);
      } else {
        // File exists but empty — initialize empty cache
        this.chatHistories.set(floorId, []);
      }
    } catch {
      // File doesn't exist yet — start fresh
      this.chatHistories.set(floorId, []);
    }
  }
```

Replace with:
```typescript
  /**
   * Clear chat history for a floor — both in-memory and in Supabase.
   *
   * Safeguard: requires explicit confirmation string to prevent accidental
   * bulk deletion. Callers must pass `confirm: 'DELETE_CHAT_HISTORY'`.
   * The floor must exist and belong to this orchestrator instance.
   */
  async clearFloorChat(floorId: string, confirm?: string): Promise<boolean> {
    // Safeguard: require explicit confirmation to prevent accidental wipes
    if (confirm !== 'DELETE_CHAT_HISTORY') {
      console.warn(`[Chat] clearFloorChat called for ${floorId} without confirmation — blocked`);
      return false;
    }
    // Safeguard: floor must exist in this orchestrator
    if (!this.floors.has(floorId)) {
      console.warn(`[Chat] clearFloorChat called for unknown floor ${floorId} — blocked`);
      return false;
    }
    const floor = this.floors.get(floorId)!;
    console.log(`[Chat] Clearing chat history for floor "${floor.name}" (${floorId})`);
    this.chatHistories.delete(floorId);
    return clearChatMessages(floorId);
  }
```
