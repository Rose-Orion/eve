## Diagnosis

The AGENT PERFORMANCE panel shows "No data yet" for Phase 8 (SideQuest) because `improvementEngine.recordTaskCompletion()` is called in `dispatchVirtual()` but the dashboard query powering the AGENT PERFORMANCE panel queries a different data structure than what `recordTaskCompletion` writes to. Tracing the pipeline: tasks complete → `recordTaskCompletion` is called → but the API route serving the AGENT PERFORMANCE panel either doesn't exist, filters by wrong field names, or the `ImprovementEngine.getPerformance()` method returns data keyed by `agentId/taskType` without any `floor`/`phase` filter — so Phase 8 SideQuest data exists in the performance store but the dashboard query can't find it because it's filtering by `floor='SideQuest'` and `phase=8` fields that don't exist on the performance records.

The root fix requires: (1) adding `floorId` and `phaseNumber` to the performance record written by `recordTaskCompletion`, (2) adding a filtered API endpoint that accepts `floorId` and `phaseNumber` query params, and (3) ensuring the route is registered. Let me trace the exact gap.

Looking at the code: `getFloorStats()` is the likely data source for the AGENT PERFORMANCE panel, and it only tracks `agentTasks` (count of completed tasks per agent) — it has **no per-agent performance metrics** (retry counts, execution time, success rate, task type). The `improvementEngine.getPerformance(floorId)` returns the right shape but there's **no API endpoint** that exposes it filtered by floor. The `floorManagerReview` shows `getPerformance(floorId)` exists but it's only used internally. The dashboard has no route to query agent performance metrics.

```typescript
import type { FastifyInstance } from 'fastify';
import type { Orchestrator } from '../../orchestrator/index.js';

export function registerTaskRoutes(app: FastifyInstance, orchestrator: Orchestrator): void {
  // Agent performance metrics — powers the AGENT PERFORMANCE panel.
  // Filters by floorId and optionally by phaseNumber so the dashboard can scope
  // results to a specific floor+phase combination (e.g. SideQuest Phase 8).
  app.get<{
    Querystring: { floorId?: string; phase?: string; floorName?: string };
  }>('/api/agent-performance', async (request, reply) => {
    const { floorId: qFloorId, phase, floorName } = request.query;

    // Resolve floorId from name if provided (dashboard may pass either)
    let resolvedFloorId = qFloorId;
    if (!resolvedFloorId && floorName) {
      const floor = orchestrator.getFloorByName(floorName);
      if (!floor) return reply.code(404).send({ error: `Floor not found: ${floorName}` });
      resolvedFloorId = floor.id;
    }
    if (!resolvedFloorId) {
      return reply.code(400).send({ error: 'floorId or floorName query param required' });
    }

    const floor = orchestrator.getFloor(resolvedFloorId);
    if (!floor) return reply.code(404).send({ error: 'Floor not found' });

    const phaseNumber = phase ? parseInt(phase, 10) : null;

    // Pull base performance records from ImprovementEngine (keyed by agentId+taskType)
    const perfRecords = orchestrator.getAgentPerformance(resolvedFloorId);

    // Pull raw task data to compute per-phase metrics and fill gaps where
    // ImprovementEngine has no record yet (tasks that completed before metrics were wired)
    const allTasks = orchestrator.getFloorTasks(resolvedFloorId);
    const scopedTasks = phaseNumber != null
      ? allTasks.filter(t => t.phaseNumber === phaseNumber)
      : allTasks;

    const completedTasks = scopedTasks.filter(t => t.status === 'completed');

    if (completedTasks.length === 0 && perfRecords.length === 0) {
      return reply.code(200).send({
        floorId: resolvedFloorId,
        floorName: floor.name,
        phaseNumber,
        records: [],
        taskCount: 0,
        message: 'No completed tasks found for this floor/phase combination.',
      });
    }

    // Build a map from task-level data (source of truth for phase-scoped queries)
    // keyed by agentId+taskType so we can merge with ImprovementEngine records
    const taskMetricMap = new Map<string, {
      agentId: string;
      taskType: string;
      phaseNumber: number;
      totalTasks: number;
      successCount: number;
      retryCount: number;
      totalRetries: number;
      avgExecutionMs: number | null;
      taskIds: string[];
    }>();

    for (const task of completedTasks) {
      const key = `${task.assignedAgent}::${task.taskType}`;
      const existing = taskMetricMap.get(key);
      const execMs = (task.dispatchedAt && task.completedAt)
        ? task.completedAt.getTime() - task.dispatchedAt.getTime()
        : null;

      if (!existing) {
        taskMetricMap.set(key, {
          agentId: task.assignedAgent,
          taskType: task.taskType,
          phaseNumber: task.phaseNumber,
          totalTasks: 1,
          successCount: 1,
          retryCount: task.attempts > 0 ? 1 : 0,
          totalRetries: task.attempts,
          avgExecutionMs: execMs,
          taskIds: [task.id],
        });
      } else {
        existing.totalTasks += 1;
        existing.successCount += 1;
        existing.totalRetries += task.attempts;
        if (task.attempts > 0) existing.retryCount += 1;
        if (execMs !== null) {
          existing.avgExecutionMs = existing.avgExecutionMs !== null
            ? Math.round((existing.avgExecutionMs + execMs) / 2)
            : execMs;
        }
        existing.taskIds.push(task.id);
      }
    }

    // Merge ImprovementEngine records (have first-try approval rate and slop violations)
    // with task-level data (have phase scope and execution time)
    const perfByKey = new Map(perfRecords.map(p => [`${p.agentId}::${p.taskType}`, p]));

    const records = [...taskMetricMap.values()].map(m => {
      const eng = perfByKey.get(`${m.agentId}::${m.taskType}`);
      const successRate = m.totalTasks > 0
        ? Math.round((m.successCount / m.totalTasks) * 100)
        : 0;
      const firstTryRate = eng && eng.totalTasks > 0
        ? Math.round((eng.firstTryApprovals / eng.totalTasks) * 100)
        : (m.totalRetries === 0 ? 100 : Math.round(((m.totalTasks - m.retryCount) / m.totalTasks) * 100));

      return {
        agentId: m.agentId,
        taskType: m.taskType,
        phaseNumber: m.phaseNumber,
        totalTasks: m.totalTasks,
        successCount: m.successCount,
        successRate,
        retryCount: m.retryCount,
        totalRetries: m.totalRetries,
        avgRetriesPerTask: m.totalTasks > 0
          ? Math.round((m.totalRetries / m.totalTasks) * 10) / 10
          : 0,
        firstTrySuccessRate: firstTryRate,
        avgExecutionMs: m.avgExecutionMs,
        avgExecutionSec: m.avgExecutionMs !== null
          ? Math.round(m.avgExecutionMs / 100) / 10
          : null,
        slopViolations: eng?.slopViolations ?? 0,
        taskIds: m.taskIds,
      };
    }).sort((a, b) => a.agentId.localeCompare(b.agentId));

    return reply.code(200).send({
      floorId: resolvedFloorId,
      floorName: floor.name,
      phaseNumber,
      records,
      taskCount: completedTasks.length,
      totalAgents: records.length,
      queriedAt: new Date().toISOString(),
    });
  });

```

**Root cause summary:** The AGENT PERFORMANCE panel had no backend route to query performance data filtered by floor and phase.
