## Diagnosis

The metrics pipeline has three distinct bugs: (1) The `task:completed` event handler in `setupEventHandlers` never emits a metrics event to any aggregator — it only handles dependency cascades and phase completion checks; (2) the `recordTaskCompletion` call in `dispatchVirtual` records performance data to the in-memory `ImprovementEngine` but there is no code path that persists `agent_performance` records to Supabase; (3) the cost calculation in `dispatchVirtual` records `result.costCents` to `task.actualCostCents` via `recordResult`, but `saveCostEvent` is only called when `result.costCents > 0` — if the Anthropic client returns zero cost due to a silent estimation failure, cost events are silently dropped. The `$4 for 10 tasks` anomaly is consistent with most cost events being dropped or the cost persisted to `cost_events` table not being aggregated into the dashboard query.

The fix needs to: (a) add explicit metrics emission in the `task:completed` handler, (b) add a Supabase persistence call for agent performance records after each task completion, and (c) add logging to surface silent cost failures.

  /**
   * Persist a completed task's agent performance record to the Supabase
   * `agent_performance` table. Called from the `task:completed` event handler
   * so it fires for all completion paths.
   *
   * Gracefully handles missing table (logs warning, does not throw) so that
   * the core task completion pipeline is never blocked by a metrics write failure.
   */
  private async persistAgentPerformanceRecord(task: Task): Promise<void> {
    try {
      const { getSupabase } = await import('../integrations/supabase.js');
      const sb = getSupabase();
      if (!sb) {
        console.warn(
          `[Metrics] Supabase client unavailable — agent performance record NOT persisted ` +
          `for task ${task.id.slice(0, 8)} (${task.taskType})`,
        );
        return;
      }

      const floor = this.floors.get(task.floorId);
      const record = {
        task_id: task.id,
        floor_id: task.floorId,
        floor_name: floor?.name ?? null,
        agent_id: task.assignedAgent,
        task_type: task.taskType,
        phase_number: task.phaseNumber,
        model_tier: task.modelTier,
        cost_cents: task.actualCostCents,
        attempts: task.attempts,
        first_try_success: task.attempts === 0,
        completed_at: task.completedAt?.toISOString() ?? new Date().toISOString(),
        created_at: task.createdAt.toISOString(),
        dispatched_at: task.dispatchedAt?.toISOString() ?? null,
        result_length: task.result?.length ?? 0,
      };

      console.log(
        `[Metrics] Persisting agent performance record: ` +
        `task=${task.id.slice(0, 8)}, agent=${task.assignedAgent}, ` +
        `type=${task.taskType}, cost=${task.actualCostCents}¢, ` +
        `phase=${task.phaseNumber}, floor="${floor?.name ?? task.floorId.slice(0, 8)}"`,
      );

      const { error } = await sb
        .from('agent_performance')
        .insert(record);

      if (error) {
        // Check for missing table (common if schema migration hasn't run)
        if (error.code === '42P01' || error.message?.includes('relation') || error.message?.includes('does not exist')) {
          console.warn(
            `[Metrics] agent_performance table does not exist — skipping persistence. ` +
            `Run schema migration to create the table. Task: ${task.id.slice(0, 8)}`,
          );
        } else {
          // Real error — log with full details for debugging
          console.error(
            `[Metrics] agent_performance INSERT failed for task ${task.id.slice(0, 8)}: ` +
            `code=${error.code}, message=${error.message}, ` +
            `hint=${error.hint ?? 'none'}, details=${error.details ?? 'none'}`,
          );
        }
      } else {
        console.log(
          `[Metrics] ✓ Agent performance record persisted: ` +
          `task=${task.id.slice(0, 8)}, agent=${task.assignedAgent}, ` +
          `cost=${task.actualCostCents}¢`,
        );
      }
    } catch (err) {
      // Never let metrics persistence crash the orchestrator
      console.error(
        `[Metrics] persistAgentPerformanceRecord threw for task ${task.id.slice(0, 8)}: ` +
        `${(err as Error).message}`,
      );
    }
  }

  // --- System Review Log ---
