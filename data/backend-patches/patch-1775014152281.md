## Diagnosis

The floor name-update flow has a backend-only bug. The `PATCH /api/floors/:id` route correctly passes the body to `orchestrator.updateFloor()`, and `updateFloor()` does mutate `floor.name` in memory and calls `persistWithRetry(() => saveFloor(floor))` — but for the specific floor `b090c7ea-ef9c-4bb3-abdf-62897e1224e8`, there's no broadcast after a name update, meaning the UI never receives a realtime event to re-render the updated name. Additionally, the Floor Manager chat has no intent-parsing for rename commands — it goes straight to OpenClaw or Claude without any pre-processing to detect rename intents and call the update endpoint.

Two fixes are needed:
1. **Backend route**: After a successful name update in `PATCH /api/floors/:id`, broadcast a `floor:updated` event so the dashboard re-renders.
2. **Chat route**: Pre-process incoming messages for rename intents before dispatching to OpenClaw/Claude, then call `updateFloor` directly and confirm to the user.

Replace with:
```typescript
  // PATCH — update floor settings
  app.patch<{
    Params: { id: string };
    Body: { name?: string; budgetCeilingCents?: number; brandState?: string; selectedBrand?: { index: number; name: string; tagline: string; personality: string; voiceAttributes: string[] } };
  }>('/api/floors/:id', async (request, reply) => {
    const floorId = request.params.id;
    const updates = request.body as Record<string, unknown>;
    const result = orchestrator.updateFloor(floorId, updates);
    if (!result) return reply.code(404).send({ error: 'Floor not found' });

    // Broadcast a realtime event so the dashboard re-renders immediately after any update.
    // This is critical for name changes — without it the UI never sees the new name.
    if (typeof updates.name === 'string') {
      const { broadcastFloorEvent } = await import('../../integrations/supabase.js');
      broadcastFloorEvent(floorId, 'floor:updated', {
        floorId,
        name: result.name,
        updatedField: 'name',
      }).catch(() => {});
    }

    return result;
  });
```

```typescript
  async sendChatMessage(floorId: string, message: string): Promise<{ response: string }> {
    const floor = this.floors.get(floorId);
    if (!floor) return { response: 'Floor not found' };

    // Load from DB only if not already loaded this call chain.
    // sendChatMessage reuses the in-session cache for the duration of
    // a single message exchange (user msg → persist → AI response → persist).
    // getChatHistory always bypasses this cache for fresh reads.
    if (!this.chatHistories.has(floorId)) {
      await this.loadChatHistory(floorId);
    }
    if (!this.chatHistories.has(floorId)) this.chatHistories.set(floorId, []);
    const history = this.chatHistories.get(floorId)!;
    const userTimestamp = new Date();
    history.push({ role: 'user', content: message, timestamp: userTimestamp });
    // Persist user message to Supabase immediately (before dispatch, so it's never lost)
    await this.persistChatMessage(floorId, 'user', message, 'owner');

    // --- Rename-intent detection ---
    // Parse common patterns: "rename to X", "change name to X", "call it X",
    // "rename this floor to X", "update name to X", "name it X"
    const renameMatch = message.match(
      /(?:rename(?:\s+(?:this\s+)?(?:floor|it))?|change\s+(?:the\s+)?name(?:\s+of\s+(?:this\s+)?floor)?|call\s+(?:this\s+(?:floor\s+)?)?it|update\s+(?:the\s+)?name|name\s+(?:this\s+(?:floor\s+)?)?it)\s+(?:to\s+)?["']?([A-Za-z0-9][A-Za-z0-9 _\-'&.]{0,60}?)["']?\s*[.!?]?\s*$/i,
    );

    if (renameMatch?.[1]) {
      const newName = renameMatch[1].trim().replace(/["'.!?]+$/, '').trim();
      const oldName = floor.name;

      if (newName && newName.length >= 1 && newName !== oldName) {
        const updated = this.updateFloor(floorId, { name: newName });
        let response: string;

        if (updated) {
          // Broadcast realtime event so the dashboard header re-renders immediately
          broadcastFloorEvent(floorId, 'floor:updated', {
            floorId,
            name: newName,
            updatedField: 'name',
          }).catch(() => {});

          response = `Done! I've renamed this floor from "${oldName}" to "${newName}". The name has been saved and the dashboard will update momentarily.`;
          console.log(`[FloorManager] Renamed floor "${oldName}" → "${newName}" (id: ${floorId}) via chat`);
        } else {
          response = `I wasn't able to rename the floor right now. Please try again or update the name from the floor settings.`;
          console.warn(`[FloorManager] Rename failed for floor ${floorId}: updateFloor returned null`);
        }

        const assistantTimestamp = new Date();
        history.push({ role: 'assistant', content: response, timestamp: assistantTimestamp });
        await this.persistChatMessage(floorId, 'assistant', response, 'floor-manager');
        return { response };
      }
    }

    // Try OpenClaw Floor Manager first
```
