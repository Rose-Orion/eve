## Diagnosis

The brand context object (`selectedBrand`, `brandState`) is passed from `Orchestrator.dispatchVirtual()` into `VirtualDispatcher.dispatch()`, but the `VirtualDispatcher` never forwards `selectedBrand` into `PromptBuilder.build()`. The `BuildPromptInput` interface accepts `selectedBrand` but `VirtualDispatcher` doesn't populate it. Separately, `seedNextPhaseTasks()` only injects brand context into the task `prompt` string for phase 4+, but phases 5 and 7 tasks and any dynamically-created tasks omit it. There is also no guard warning when a task is dispatched without brand context while a brand exists on the floor.

```typescript
  private async buildBrandSection(
    floorSlug: string,
    agentId: AgentId,
    brandState: BrandState,
    template: AgentTemplate,
    selectedBrand: import('../config/types.js').SelectedBrand | null = null,
  ): Promise<PromptSection | null> {
    if (brandState === 'pre-foundation') {
      const preFoundation = getPreFoundationContext(agentId);
      if (!preFoundation) return null;
      return {
        tag: 'brand_context',
        content: preFoundation,
        tokenCount: await countTokens(preFoundation),
        priority: TOKEN_BUDGET.sections.brandContext.priority,
      };
    }

    const brand = await loadBrandContext(floorSlug, agentId, brandState);
    // Build XML from file-based context (may be null pre-phase-4)
    let xml = brand ? formatBrandContextXml(brand) : '';

    // Always inject owner-selected brand direction when available —
    // this is the authoritative brand signal and must be present regardless
    // of whether the file-based brand context has been written yet.
    if (selectedBrand) {
      const selectionBlock = `<selected_brand_direction>
OWNER-SELECTED BRAND DIRECTION (authoritative — use this name and personality for ALL work):
Brand Name: ${selectedBrand.name}
Tagline: "${selectedBrand.tagline}"
Personality: ${selectedBrand.personality}
Voice Attributes: ${selectedBrand.voiceAttributes?.join(', ') || 'Not specified'}

CRITICAL: The owner chose this specific brand direction. Every deliverable must use this brand name and align with this personality and voice.
</selected_brand_direction>`;

      xml = xml
        ? `${selectionBlock}\n\n${xml}`
        : selectionBlock;
    }

    if (!xml) return null;

    return {
      tag: 'brand_context',
      content: xml,
      tokenCount: await countTokens(xml),
      priority: TOKEN_BUDGET.sections.brandContext.priority,
    };
  }
```
