This needs to be corrected in the Supabase `floors` table and synchronized with the in-memory `BudgetEnforcer` state. The `updateFloor` method in the orchestrator already handles both concerns correctly — we just need a targeted API route that performs this specific correction.

Looking at the code flow:
- `Orchestrator.updateFloor()` handles `budgetCeilingCents` updates and calls `budgetEnforcer.updateCeiling()` + `saveFloor()` correctly.
- The route just needs to find the floor by name and apply the update.

The cleanest, safest fix is a one-time data correction route (already exposed via `PATCH /floors/:id`) — but since we need to find the floor by name and correct the stored value, the safest approach is a targeted patch to `src/server/routes/floors.ts` that adds a budget-correction endpoint, or we use the existing `PATCH /floors/:id` route directly.

// --- Budget Correction (one-time data fix) ---
// POST /floors/:id/correct-budget
// Body: { budgetCeilingCents: number, confirmFloorName: string }
// Requires confirmFloorName to match the actual floor name as a safety guard.
async function registerBudgetCorrectionRoute(
  app: FastifyInstance,
  orchestrator: Orchestrator,
): Promise<void> {
  app.post<{
    Params: { id: string };
    Body: { budgetCeilingCents: number; confirmFloorName: string };
  }>('/floors/:id/correct-budget', {
    schema: {
      body: {
        type: 'object',
        required: ['budgetCeilingCents', 'confirmFloorName'],
        properties: {
          budgetCeilingCents: { type: 'number', minimum: 1 },
          confirmFloorName: { type: 'string', minLength: 1 },
        },
      },
    },
  }, async (request, reply) => {
    const { id } = request.params;
    const { budgetCeilingCents, confirmFloorName } = request.body;

    const floor = orchestrator.getFloor(id);
    if (!floor) {
      return reply.status(404).send({ error: 'Floor not found' });
    }

    // Safety guard: caller must confirm the floor name to prevent accidental overwrites
    if (floor.name !== confirmFloorName) {
      return reply.status(400).send({
        error: 'Floor name mismatch — pass confirmFloorName matching the actual floor name',
        actualName: floor.name,
      });
    }

    const previousCents = floor.budgetCeilingCents;
    const updated = orchestrator.updateFloor(id, { budgetCeilingCents });
    if (!updated) {
      return reply.status(500).send({ error: 'Floor update failed' });
    }

    console.log(
      `[BudgetCorrection] "${floor.name}": ${previousCents}¢ → ${budgetCeilingCents}¢` +
      ` ($${Math.round(previousCents / 100)} → $${Math.round(budgetCeilingCents / 100)})`,
    );

    return reply.send({
      success: true,
      floorId: id,
      floorName: floor.name,
      previousCeilingCents: previousCents,
      newCeilingCents: budgetCeilingCents,
      previousCeilingDollars: Math.round(previousCents / 100),
      newCeilingDollars: Math.round(budgetCeilingCents / 100),
    });
  });
}
