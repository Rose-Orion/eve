## Diagnosis

The investigation requires: (1) confirming `brand-details` is a downstream dependency of `budget-plan` in the dependency config; (2) verifying the `budget-plan` completion event fired and was received by the queue/trigger service; (3) manually enqueuing `brand-details` if it was defined but not queued; (4) adding a diagnostic log/guard to ensure task-completion events always trigger downstream dependency checks.

Reading the codebase: Phase 3 (Foundation Sprint) seeds `brand-options`, `business-strategy`, and `budget-plan` — but there is **no `brand-details` task** defined anywhere in Phase 3 or Phase 4. The `brand-details` deliverable appears to be a downstream conceptual artifact, but no task with `taskType: 'brand-details'` is seeded, and no dependency relationship exists linking `budget-plan` completion → `brand-details`. The dependency graph only processes tasks via `dependsOn` arrays — since `brand-details` was never created, it was never triggered.

The second issue: `setupEventHandlers` handles `task:completed` and calls `checkPhaseCompletion`, then `dependencyGraph.onTaskCompleted`. However, there is **no diagnostic log** when dependency-ready tasks fail to transition, and no guard ensuring all `dependsOn`-linked tasks actually get queued when their upstream completes. If `brand-details` were added with `dependsOn: [budgetPlanTaskId]`, the existing flow would handle it — but only if the task exists in the first place.

**Root cause**: `brand-details` task is never seeded for Quest Kids Phase 3. Additionally, the `task:completed` event handler has no fallback log when zero downstream tasks become ready (silent gap). We need to: (A) add `brand-details` as a Phase 3 task with `dependsOn: [budget-plan task id]`; (B) add the diagnostic guard to `setupEventHandlers`; (C) add a manual enqueue method.

```typescript
  // --- Manual Downstream Task Enqueue (Public API) ---

  /**
   * Manually enqueue any downstream tasks for a given completed task.
   * Used when the completion event for a task (e.g. budget-plan) was received
   * but downstream dependencies (e.g. brand-details) were not triggered.
   *
   * Steps:
   * 1. Find the named task by taskType on this floor (must be completed)
   * 2. Force-run the dependency cascade for that task's ID
   * 3. Queue any newly-ready downstream tasks
   * 4. Return a structured report of what was enqueued
   */
  async manualEnqueueDownstreamTasks(
    floorId: string,
    upstreamTaskType: string,
  ): Promise<{
    floorId: string;
    floorName: string;
    upstreamTaskType: string;
    upstreamTaskId: string | null;
    upstreamStatus: string | null;
    downstreamEnqueued: Array<{ taskId: string; taskType: string; status: string }>;
    alreadyQueued: Array<{ taskId: string; taskType: string; status: string }>;
    enqueuedAt: string;
    message: string;
  }> {
    const enqueuedAt = new Date().toISOString();
    const floor = this.floors.get(floorId);
    const floorName = floor?.name ?? 'Unknown';

    // Find the upstream task (preferably completed)
    const allTasks = this.taskManager.getFloorTasks(floorId);
    const upstreamTask =
      allTasks.find(t => t.taskType === upstreamTaskType && t.status === 'completed') ??
      allTasks.find(t => t.taskType === upstreamTaskType);

    if (!upstreamTask) {
      return {
        floorId, floorName, upstreamTaskType,
        upstreamTaskId: null, upstreamStatus: null,
        downstreamEnqueued: [], alreadyQueued: [],
        enqueuedAt,
        message: `No task of type "${upstreamTaskType}" found for floor "${floorName}".`,
      };
    }

    if (upstreamTask.status !== 'completed') {
      return {
        floorId, floorName, upstreamTaskType,
        upstreamTaskId: upstreamTask.id, upstreamStatus: upstreamTask.status,
        downstreamEnqueued: [], alreadyQueued: [],
        enqueuedAt,
        message: `Upstream task "${upstreamTaskType}" is not completed (status=${upstreamTask.status}). ` +
          `Downstream tasks can only be enqueued after upstream completes.`,
      };
    }

    // Force-run the dependency cascade
    const newlyReady = this.dependencyGraph.onTaskCompleted(upstreamTask.id);
    const downstreamEnqueued: Array<{ taskId: string; taskType: string; status: string }> = [];
    const alreadyQueued: Array<{ taskId: string; taskType: string; status: string }> = [];

    for (const taskId of newlyReady) {
      const downTask = this.taskManager.getTask(taskId);
      if (!downTask) continue;

      if (downTask.status === 'queued' || downTask.status === 'dispatched' || downTask.status === 'working') {
        alreadyQueued.push({ taskId, taskType: downTask.taskType, status: downTask.status });
        continue;
      }

      const queued = this.taskManager.transition(taskId, 'queued');
      if (queued) {
        downstreamEnqueued.push({ taskId, taskType: downTask.taskType, status: 'queued' });
        console.log(
          `[ManualEnqueue] Queued downstream task ${taskId.slice(0, 8)} ` +
          `(${downTask.taskType}) for floor "${floorName}"`,
        );
      } else {
        console.warn(
          `[ManualEnqueue] Failed to queue ${taskId.slice(0, 8)} (${downTask.taskType}, ` +
          `status=${downTask.status}) — transition rejected`,
        );
      }
    }

    // Also check for brand-details tasks in 'created' state with all deps completed
    const brandDetailsTasks = allTasks.filter(
      t => t.taskType === 'brand-details' && t.status === 'created' &&
        t.dependsOn.every(depId => {
          const dep = this.taskManager.getTask(depId);
          return dep?.status === 'completed';
        }),
    );
    for (const bdt of brandDetailsTasks) {
      if (downstreamEnqueued.some(d => d.taskId === bdt.id) ||
          alreadyQueued.some(d => d.taskId === bdt.id)) continue;
      const queued = this.taskManager.transition(bdt.id, 'queued');
      if (queued) {
        downstreamEnqueued.push({ taskId: bdt.id, taskType: bdt.taskType, status: 'queued' });
        console.log(
          `[ManualEnqueue] Queued brand-details task ${bdt.id.slice(0, 8)} ` +
          `(deps all completed) for floor "${floorName}"`,
        );
      }
    }

    const totalEnqueued = downstreamEnqueued.length;
    const message = totalEnqueued > 0
      ? `Manually enqueued ${totalEnqueued} downstream task(s) for "${upstreamTaskType}" completion on "${floorName}".`
      : alreadyQueued.length > 0
      ? `All ${alreadyQueued.length} downstream task(s) already queued/active — no action needed.`
      : `No downstream tasks found for "${upstreamTaskType}" on "${floorName}". ` +
        `Check dependency graph registration.`;

    await this.writeSystemReviewLog({
      event: 'manual_downstream_enqueue',
      floorId, floorName, upstreamTaskType,
      upstreamTaskId: upstreamTask.id,
      downstreamEnqueued, alreadyQueued, enqueuedAt, message,
    });

    console.log(`[ManualEnqueue] ${message}`);
    return {
      floorId, floorName, upstreamTaskType,
      upstreamTaskId: upstreamTask.id,
      upstreamStatus: upstreamTask.status,
      downstreamEnqueued, alreadyQueued, enqueuedAt, message,
    };
  }

  // --- Phase Task Queue Investigation (Public API) ---
```
