## Diagnosis

The `showTaskDeliverable()` function renders task output as raw pre-formatted text in a plain scrollable box — no visual hierarchy, no metadata sections, no icons beyond the agent emoji, and no design language matching the glass/card system used elsewhere. The function needs a full redesign that structures the content into sections (summary header, status metadata, content body, council panel) using the existing design tokens and adds visual polish consistent with the rest of the dashboard.

          border:1px solid rgba(255,255,255,0.08);
