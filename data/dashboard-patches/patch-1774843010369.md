## Diagnosis

The brand visual system tokens are already defined in the CSS `:root` (brand primary `#4A3AFF`, secondary `#FFB833`, accent `#2ECC71`, gradient, glow, font stack) and partially wired to UI components, but several components still use raw hex bypasses (`#6366f1`, `#8b5cf6`, `rgba(99,102,241,...)`, etc.) instead of the brand token variables. The floor UI components (backgrounds, nav, cards, buttons, typography, progress bars, chat panel) need their hardcoded color references replaced with the established token variables. No layout, spacing, or functional changes are needed — only CSS value replacements.
