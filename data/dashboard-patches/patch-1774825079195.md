## Diagnosis

The `viewNotifications()` function contains a hardcoded `router.go('gate1', ...)` call in the "Review" button for **all** pending approval items, regardless of their type. This means a Content Production approval (phaseNumber 8, which should route to `gate3`) incorrectly navigates to the `gate1` (Foundation/Brand) view. The Tower view (`viewTower`) has correct conditional routing logic for approval types, but `viewNotifications()` bypasses it entirely.

## Patch

### Change 1: Fix hardcoded `gate1` route in `viewNotifications()` Review buttons

**File:** `public/app.js`

**What it changes:** Replaces the hardcoded `router.go('gate1', ...)` in the notifications approval list with the same conditional routing logic already used in `viewTower()` — checking `a.type`, `a.phaseNumber`, and falling back gracefully.

**Why:** Every approval item in `viewNotifications()` sends the user to `gate1` regardless of type. A Content Production approval has `phaseNumber === 8` and should route to `gate3`. A Website/Staging approval has `phaseNumber === 6` and should route to `gate2`. Only Foundation Sprint (`phaseNumber === 3`) should route to `gate1`.

**Find:**
```javascript
    ${pending.length > 0 ? `
    <div class="section-label" style="margin-bottom:12px">NEEDS YOUR APPROVAL</div>
    ${pending.map(a => `
    <div class="attention-item glass" style="margin-bottom:10px;border-radius:var(--radius)">
      <div>
        <div class="attention-text">🔴 ${a.title || 'Review pending'}</div>
        <div style="font-size:12px;color:var(--text-3);margin-top:4px">${a.description || ''}</div>
      </div>
      <button class="btn btn-primary btn-sm" onclick="router.go('gate1', {id:'${a.floorId}'})">Review →</button>
    </div>`).join('')}` : `
```

**Replace with:**
```javascript
    ${pending.length > 0 ? `
    <div class="section-label" style="margin-bottom:12px">NEEDS YOUR APPROVAL</div>
    ${pending.map(a => {
      let reviewDest;
      if (a.type === 'gate') {
        const gateView = a.phaseNumber === 3 ? 'gate1'
          : a.phaseNumber === 6 ? 'gate2'
          : a.phaseNumber === 8 ? 'gate3'
          : null;
        reviewDest = gateView
          ? `router.go('${gateView}', {id:'${a.floorId}'})`
          : `router.go('floor', {id:'${a.floorId}'})`;
      } else {
        reviewDest = `router.go('floor', {id:'${a.floorId}'})`;
      }
      return `
    <div class="attention-item glass" style="margin-bottom:10px;border-radius:var(--radius)">
      <div>
        <div class="attention-text">🔴 ${a.title || 'Review pending'}</div>
        <div style="font-size:12px;color:var(--text-3);margin-top:4px">${a.description || ''}</div>
      </div>
      <button class="btn btn-primary btn-sm" onclick="${reviewDest}">Review →</button>
    </div>`;
    }).join('')}` : `
```
